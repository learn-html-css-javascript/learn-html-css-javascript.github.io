Empty Elements

What are Empty Elements?

  • Empty elements are elements that do not have a closing tag
<!-- p tags are NOT empty elements because they have a closing tag ("</p>") -->
<p> I am not an empty element </p>


<!-- img tags are considered empty elements because they DO NOT have a closing tag -->

<img src="images/team_hands" alt="a group of people touching hands in a star formation" title="teamwork is awesome">

  • Empty elements cannot contain child elements (i.e. elements cannot be nested inside of an empty element)

  • Empty elements are sometimes referred to as “self-closing tags”

Which elements are considered Empty Elements?

The following is a list of all the Empty Elements found in HTML:

  • <area>
  • <base>
  • <br>
  • <col>
  • <embed>
  • <hr>
  • <img>
  • <input>
  • <keygen>
  • <link>
  • <meta>
  • <param>
  • <source>
  • <track>
  • <wbr>
